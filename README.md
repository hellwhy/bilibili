## 技术栈
vue + vue-router + ES6 + less + flex + vant + axios

## 功能
登录、注册、权限验证  
筛选分类  
编辑个人信息、上传头像  
下拉加载  
视频播放、推荐视频、收藏视频、发表评论  

## 项目部署
```
npm install
```

### 项目运行
```
npm run serve
```

### 项目打包
```
npm run build
```
